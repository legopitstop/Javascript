# rawjson.js
rawjson.js will take a JSON and return HTML that can then be placed in the docuemnt.

## Table of Contents
- [Install](#install)
- [JSON Schema](#json-schema)
- [rawjson](rawjson.md)

## Install
Add the following code to your header.
```html
<script src="https://cdn.jsdelivr.net/gh/legopitstop/Javascript/libs/rawjson/script.js"></script>
```

## JSON Schema
Add this JSON schema to your JSON file.
```json
{
    "$schema": "https://cdn.jsdelivr.net/gh/legopitstop/Javascript/libs/rawjson/schema.json"
}
```
