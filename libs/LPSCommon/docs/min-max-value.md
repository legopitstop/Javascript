# MinMaxValue
Checks if value is imbetween min and max values.
## Table of Contents
- [Syntax](#syntax)
- [Example](#example)
---

## Syntax
```typescript
MinMaxValue(value: Number, min: Number, max: Number): Boolean
```
- `value` - 
- `min` - 
- `max` - 
## Example
```js
var value = 3
MinMaxValue(value, 0, 3)
// returns true

var value = 5
MinMaxValue(value, 0, 3)
// returns false and sends error to console.
```
